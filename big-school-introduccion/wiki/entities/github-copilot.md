# GitHub Copilot

## Overview
Asistente de IA integrado directamente en el IDE, desarrollado por GitHub (Microsoft). Su función principal es el autocompletado inteligente de código y la protección del "Flow State" del desarrollador al eliminar la necesidad de cambiar de contexto al navegador.

## Key Information
- **Tipo:** Herramienta / Plugin de IDE
- **Desarrollador:** GitHub (Microsoft)
- **Integración:** Directamente en el editor de código (VS Code, JetBrains, etc.)
- **Capacidades principales:** Autocompletado predictivo, chat para explicaciones, refactorización, Comment-Driven Development (CDD)

## Notes
Destaca frente a Cursor en flujos de trabajo donde el desarrollador no necesita comprensión arquitectónica de proyecto completo. Copilot es excelente para velocidad táctica a nivel de archivo; Cursor para visión a nivel de proyecto.
